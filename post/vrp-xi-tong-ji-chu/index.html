VRP系统基础

2021-09-13 3 min read

管理方式

  • 本地管理：使用console线进行，使用serial协议，适合初始化、故障恢复。

  • talent/ssh：远程登录(VTY口)，后期维护，SSH/talnet协议。

认证模式

认证模式 描述
AAA 登录时需要用户名与密码
Password 登录时只需要密码

talnet配置（远程连接）

talnet server enable                            # 开启talnet服务
display telnet server status                    # 验证talnet服务
user-interface vty 0 4                          # 进入VTY配置模式
    authentication-mode password/aaa            # 配置认证模式
    set authentication password cipher XXXX     # 配置认证密码
    user privilege level xx                     # 配置用户等级
user-interface maximum-vty xx                   # 配置vty会话最大数量
#aaa配置
aaa
    local-user xxx passoword cipher xxx           # 创建用户与密码
    local-user xxx privilege level xx             # 配置用户级别 
    local-user xxx service-type (telnet)          # 配置用户可用服务
