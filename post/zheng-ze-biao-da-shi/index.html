2020-03-07 JavaScript

JavaScript 正则表达式

基本格式

  • /正则表达式/ 修饰符（可选）
  • 调用RegExp()对象的构造函数

在JavaScript中正则表达式有两种字符方法：search()和replace()

search()方法：用于检索字符串中指定的子字符串，或者检索一个与正则表达式匹配的子字符串，并返回子字符串的首位置

replace()方法：用于替换字符串中的指定的一些字符，或者换一个与正则表达式匹配的子字符串

基础元字符

元字符 说明
. 匹配除换行符以外的任意字符
\w 匹配字母或者数字或者下划线
\s 匹配任意的空白符
\d 匹配数字
\b 匹配单词的开始或者结束
^ 匹配字符串的开始
$ 匹配字符串的结束 
// 匹配有abc开头的字符串
\babc 或者^abc
// 匹配8位数字的qq号码
^\d\d\d\d\d\d\d\d$
//匹配1开头11位数字的电话号码
^1\d\d\d\d\d\d\d\d\d\d$

重复限定符

说明 语法
* 重复零次或者多次
+ 重复一次或者更多次
？ 重复零次或者一次
{n} 重复n次
{n,} 重复n次或者更多次
{n,m} 重复n到m次 
//匹配8位数字的qq号
^\d{8}$
//匹配1开头11位数字的电话号码
^1\d{10}$
//匹配银行卡卡号为14~18位的数字
^\d{14,18}$
//匹配以a开头的，0个或者多个b结尾的字符串
^ab*$

分组

正则表达式用（）来分组，括号里的内容作为一个整体

当我们需要匹配多个ab时，就可以这样：

^(ab)*

转义

当需要匹配的字符为正则表达式的关键保留字符时，通过转义来实现

当匹配（ab）开头时，需要把括号进行转义：

^(\(ab\))*

条件或

正则表达式用|来表示或关系，也叫分支关系，当满足条件的分子条件中的任何一个条件时，都会匹配成功

匹配联通号码以130、131、132、155、180开头的号码

^(130|131|132|155|180)\d{8}$

区间

正则表达式提供一个元字符[]来表示区间条件

  1. 限定0到9可以写成[0-9]
  2. 限定A-Z可以写成[A-Z]
  3. 限定某些数字可以写成[ 165]

所以上一问题可以写成

^(13[0-2]|155|180)\d{8}$

贪婪与非贪婪

  • 贪婪匹配会尽可能多的匹配正则字符串

    /a+/去匹配'aaaa'时会尽可能多的匹配，结果为：
'aaaa'

  • 非贪婪匹配会尽可能少的匹配

    /a+?/去匹配'aaaa'时，结果为
'a'

修饰符

在正则表达式的第二条/后可以跟修饰符（一个或者多个）

常用修饰符：

修饰符 作用
i 执行不区分大小写的匹配
g 全局匹配
m 多行匹配模式
y 粘连修饰符：与g相似都是全局匹配
s 表示(.)表示任意字符，不设置的话，四个字节的utf-16字符和航中字符不能表示
u 开启Unicode模式，正确处理大于\uFFFF的Unicode字符

RegExp属性

  • flags会返回正则表达式的修饰符
  • source只读字符串，包含正则表达式的文本
  • lastIndex可读/可写整数、

RegExp的方法

exec():

如果没有找到任何属性，返回 null 如果找到匹配返回一个数组，该数组的第一个元素是相匹配的字符串，余下的元素是与圆括号内的子表达式相匹配的子串

test()：

调用test()与exec()等价，当exec()返回值不是 null ，test() 返回 true ，否则返回 false 。
